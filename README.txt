Setting up project on local system:

Prerequisites
•Java JDK is installed (http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
•IntelliJ IDEA "Community" is installed (https://www.jetbrains.com/idea/download/#section=windows)
•Git is installed (https://git-for-windows.github.io/)
•Set JAVA_HOME variable (https://www.mkyong.com/java/how-to-set-java_home-on-windows-10/)


 Steps to import and setup project
•Open IntelliJ IDEA -> "Check out from Version Control" -> Git (Git repository URL: http://msplap318:7990/scm/cl-ta/patient_web_app.git)   and click "Clone" button.
•Wait until all dependencies are downloaded.
•Open File -> Project Structure and set path to jdk (C:\Program Files\Java\jdk1.8.0_144) in Project SDK.
•Open File -> Settings -> Plugins -> Browse repositories...
•Install Cucumber for Java plugin
•Install Gherkin plugin
•Restart IntelliJ IDEA
